# ⚔️ 카카오톡 '검키우기' 자동화 스크립트 가이드

본 가이드는 `target_20.py` 스크립트를 활용하여 카카오톡 게임을 자동화하는 설정 및 실행 방법을 안내합니다.

---

## 1. 사전 환경 설정 (OS별 코드 수정)
사용 중인 운영체제(OS)에 따라 단축키 설정을 변경해야 합니다.

* **대상 파일:** `target_20.py`
* **수정 위치:** 소스코드 **98 라인**
* **설정 방법:**
    * **Windows:** `command`를 **`ctrl`**로 변경
    * **Mac:** 기존 설정(`command`) 그대로 유지

---

## 2. 좌표 파악 및 설정 (Configuration)
스크립트가 클릭할 정확한 위치를 지정하는 단계입니다.

1.  **카카오톡 실행:** '검키우기'를 진행할 채팅방 창을 화면에 띄웁니다.
2.  **좌표 확인:** `pointer_loc_check.py`를 실행하여 마우스 포인터가 위치한 **x, y 좌표**를 확인합니다.
3.  **코드 반영:** 확인한 좌표값을 `target_20.py`의 **88번(x값), 89번(y값)** 라인에 각각 입력하고 저장합니다.

---

## 3. 스크립트 실행 및 동작 로직
모든 설정이 완료되면 아래 순서대로 자동화를 시작합니다.

1.  **준비:** 카카오톡 채팅창이 다른 창에 가려지지 않도록 화면 전면에 배치합니다.
2.  **실행:** `target_20.py` 파일을 실행합니다.
3.  **자동화 동작:**
    * 스크립트가 화면에서 **'강화'** 이미지를 검색합니다.
    * **[이미지 발견 시]:** 해당 버튼 위치를 자동으로 클릭합니다.
    * **[이미지 미발견 시]:** 채팅창에 직접 **`/강화`**를 입력하여 명령을 수행합니다.

---

> **💡 참고 사항**
> * 카카오톡 창의 크기가 바뀌거나 위치가 이동하면 좌표를 다시 설정해야 합니다.
> * 이미지 인식률을 높이기 위해 카카오톡 테마는 기본(기본 화이트/다크)을 권장합니다.
